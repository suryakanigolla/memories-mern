## About The Project





### Built With

* [ReactJS](https://reactjs.org/)
* [Mongo DB ](https://www.mongodb.com/)
* [Express JS ](https://expressjs.com/)
* [Node JS](https://nodejs.org/en/)



### Installation
   ```sh
   git clone https://github.com/suryakanigolla/memories-mern.git .
   npm run build
   npm run start
   ```
